# Лабораторная работа №6

## Тема
Сложные анимации, векторная графика и кастомные интерактивные действия (интерактивная таблица оценок).

## Цель работы
Научиться создавать сложные анимации, работать с векторной графикой и кастомными интерактивными действиями.

## Постановка задачи
Разработать таблицу оценок с:
- анимированными строками при перестановке мест;
- SVG-прогресс-барами успеваемости;
- интерактивными эффектами наведения на ячейки оценок.

## Реализация
- Использован `Vue 3` (CDN) и `Tailwind CSS`.
- Реализована композиция компонентов с передачей данных через `props`:
  - `SummaryCards` — статистика по группе;
  - `SortControls` — кнопки сортировки/перестановки;
  - `ScoreTable` — таблица оценок, SVG-прогресс и интерактивные ячейки.
- Анимация перестановки строк:
  - `transition-group` для `tbody`;
  - CSS-класс `.row-move` (FLIP-анимация перемещения строк).
- SVG-графика с анимацией:
  - круговые progress-rings в каждой строке;
  - управление `stroke-dasharray` и `stroke-dashoffset`;
  - повторный запуск анимации при сортировке (`animateRings`).
- Кастомные интерактивные действия:
  - эффект «светового пятна» в ячейках оценок по движению указателя (`pointermove`);
  - возврат в центр при `pointerleave`;
  - click-анимация ячейки через `element.animate(...)`.

## Файлы работы
- `index.html` — структура страницы, подключение Tailwind/Vue и компонентная разметка.
- `script.js` — данные, компоненты, сортировки, анимации и интерактивная логика.
