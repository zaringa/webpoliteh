# Лабораторная работа №3

## Тема
Управляющие конструкции для отображения списков и асинхронных данных (каталог онлайн-библиотеки).

## Цель работы
Научиться работать с управляющими конструкциями для отображения списков и асинхронных данных.

## Постановка задачи
Создать каталог книг онлайн-библиотеки с возможностью:
- фильтрации по статусу наличия;
- асинхронной загрузки списка книг из имитации сервера;
- отображения состояния загрузки и доступности книг.

## Реализация
- Использован `Vue 3` (CDN) и `Tailwind CSS`.
- Реализованы компоненты с передачей свойств (`props`):
  - `LibraryStats` — отображение счётчиков (всего, доступно, выдано);
  - `FilterTabs` — фильтры списка книг и отображение количества по каждому фильтру;
  - `BookGrid` — вывод карточек книг.
- Использованы управляющие конструкции:
  - `v-for` для вывода фильтров и списка книг;
  - `v-if / v-else-if / v-else` для состояния загрузки, ошибки, пустого фильтра и успешного списка.
- Асинхронная загрузка:
  - функция `fetchBooksFromServer()` имитирует ответ сервера через `Promise + setTimeout`;
  - `loadBooks()` обрабатывает `loading`, `success`, `error`.
- Фильтрация:
  - `all` — все книги;
  - `available` — только доступные;
  - `unavailable` — только выданные.

## Файлы работы
- `index.html` — структура страницы и подключение Vue/Tailwind.
- `script.js` — компоненты, реактивное состояние, асинхронная загрузка и фильтрация.

4. Использование переданных свойств компонентов: выполнено (`props` в `LibraryStats`, `FilterTabs`, `BookGrid`).
