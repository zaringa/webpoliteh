# Лабораторная работа №4

## Тема
Обработка пользовательских событий и двусторонняя привязка данных формы (форум читателей).

## Цель работы
Освоить обработку пользовательских событий и двусторонние привязки данных форм.

## Постановка задачи
Разработать форму публикации комментариев для форума читателей с возможностью:
- отметки уведомлений;
- автоматической загрузки существующих комментариев при открытии формы;
- публикации новых комментариев с валидацией полей.

## Реализация
- Использован `Vue 3` (CDN) и `Tailwind CSS`.
- Интерфейс выполнен в упрощённом тёмном стиле с фиолетовыми акцентами.
- Двусторонние привязки реализованы через `v-model` для всех элементов формы:
  - `input` (имя, книга);
  - `textarea` (текст комментария);
  - `select` (оценка);
  - `checkbox` (уведомления).
- Обработка пользовательских событий:
  - кнопка публикации (`submitComment`);
  - кнопка заполнения примера (`fillDemo`);
  - кнопка очистки (`resetForm`);
  - кнопка повторной загрузки комментариев (`loadComments`).
- Предотвращение стандартного поведения формы:
  - `@submit.prevent`;
  - `@reset.prevent`.
- Жизненный цикл компонента:
  - автоматическая инициализация данных в `onMounted()`;
  - загрузка существующих комментариев из имитации сервера (`Promise + setTimeout`).

## Файлы работы
- `index.html` — структура страницы и подключение Vue/Tailwind.
- `script.js` — реактивное состояние, обработка событий, автозагрузка комментариев.

1. Корректная обработка событий кнопок и полей: выполнено.
2. Двусторонние привязки всех элементов формы: выполнено (`v-model` на всех полях).
3. Правильная работа жизненного цикла компонента: выполнено (`onMounted`).
4. Предотвращение стандартного поведения форм: выполнено (`@submit.prevent`, `@reset.prevent`).
5. Отчет оформлен согласно требованиям: выполнено (данный файл).

7. Работа защищена очно преподавателю: выполняется студентом.
