# Лабораторная работа №5

## Тема
Управление глобальным состоянием приложения и плавные анимационные переходы (корзина интернет-магазина).

## Цель работы
Изучить управление глобальным состоянием приложения и создание плавных анимационных переходов.

## Постановка задачи
Создать корзину интернет-магазина, где:
- товары добавляются в глобальное состояние;
- подсчитывается общая стоимость и количество;
- корзина плавно появляется/скрывается;
- счетчик товаров анимируется и синхронизируется с состоянием.

## Реализация
- Использован `Tailwind CSS` для интерфейса и анимационных классов.
- Глобальное состояние реализовано через единый объект `state`:
  - `state.cart` — покупки;
  - `state.isCartOpen` — видимость корзины.
- Действия с состоянием:
  - `addToCart(productId)` — добавить товар;
  - `clearCart()` — очистить корзину;
  - `toggleCart()` / `closeCart()` — управление панелью корзины.
- Производные данные:
  - `getCartItems()` формирует список позиций;
  - `getTotals()` вычисляет итоговое количество и общую стоимость.
- Плавные анимации:
  - открытие/закрытие корзины через `translate-x`, `opacity`, `transition`;
  - анимация счетчика товаров через `animate-badge-pop`.
- Синхронизация интерфейса:
  - единый `render()` обновляет каталог, корзину, счетчик и итоговую стоимость.

## Файлы работы
- `index.html` — структура страницы, Tailwind и панель корзины.
- `script.js` — глобальное состояние, обработка действий, вычисление итогов и анимации.

## Проверка критериев
1. Корректная работа глобального состояния: выполнено (`state`, функции изменения состояния).
2. Вычисление производных данных (общая стоимость): выполнено (`getTotals`).
3. Плавные анимационные переходы: выполнено (панель корзины и бейдж счетчика).
4. Синхронизация счетчика с состоянием: выполнено (обновление в `renderTotals`).
5. Отчет оформлен согласно требованиям: выполнено (данный файл).
6. Отчет загружен в СДО: выполняется студентом.
7. Работа защищена очно преподавателю: выполняется студентом.
8. Обязательное условие для профиля дизайн (Tailwind CSS): выполнено.
