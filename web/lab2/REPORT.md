# Лабораторная работа №2

## Тема
Создание компонентов и реактивного отображения данных (блок новостей информационного портала).

## Цель работы
Освоить базовые принципы создания компонентов и реактивного отображения данных.

## Постановка задачи
Разработать блок новостей информационного портала, где:
- заголовок и содержимое обновляются при изменении категории новостей;
- реализовано динамическое выделение топ-новостей с помощью условных стилей.

## Реализация
- Использован `Vue 3` (CDN) для реактивного состояния и компонентного подхода.
- Использован `Tailwind CSS` для интерфейса и адаптивной вёрстки.
- Созданы компоненты:
  - `CategoryTabs` — переключение категорий;
  - `NewsList` — вывод списка материалов и выбор активной новости.
- Реактивное состояние:
  - `activeCategoryId` — выбранная категория;
  - `activeArticleId` — выбранная новость.
- При смене категории автоматически:
  - обновляется заголовок раздела;
  - пересчитывается и отображается лид-материал;
  - обновляется список новостей категории.
- Условные стили:
  - новости с флагом `top: true` получают выделение (`TOP`-badge, рамка/фон);
  - активный материал дополнительно выделяется контрастной карточкой.

## Файлы работы
- `index.html` — структура страницы, подключение Tailwind/Vue.
- `script.js` — данные, компоненты, реактивная логика.

1. Корректное отображение динамического контента: выполнено.
2. Реактивное обновление при смене категории: выполнено.
3. Правильное применение условных стилей: выполнено.
