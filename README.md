**Дано:**  
(A={a,e,f,k,t}),  
(B={f,i,j,p,y}),  
(C={j,k,l,y}),  
(D={i,j,s,t,u,y,z}).

Найти мощности множеств:

[  
X=(A\cap B)\cup(D\cap C),  
\qquad  
Y=(A\cap \overline B)\cup(D\setminus C).  
]

**Решение.**

[  
A\cap B={f}.  
]

[  
D\cap C={j,y}.  
]

Тогда

[  
X={f}\cup{j,y}={f,j,y}.  
]

[  
|X|=3.  
]

Теперь найдем (Y).

[  
A\cap \overline B=A\setminus B={a,e,k,t}.  
]

[  
D\setminus C={i,s,t,u,z}.  
]

Тогда

[  
Y={a,e,k,t}\cup{i,s,t,u,z}  
={a,e,i,k,s,t,u,z}.  
]

[  
|Y|=8.  
]

**Ответ:**

[  
|X|=3,\qquad |Y|=8.

## Задание 3

Проиллюстрировать равенство диаграммами Эйлера-Венна:

[  
(A\setminus B)\cap(A\setminus C)=A\setminus(B\cup C).  
]

**Решение.**

Преобразуем левую часть:

[  
A\setminus B=A\cap \overline B,  
]

[  
A\setminus C=A\cap \overline C.  
]

Тогда

[  
(A\setminus B)\cap(A\setminus C)  
=(A\cap \overline B)\cap(A\cap \overline C).  
]

[  
= A\cap \overline B\cap \overline C.  
]

По закону де Моргана:

[  
\overline B\cap \overline C=\overline{B\cup C}.  
]

Следовательно,

[  
A\cap \overline B\cap \overline C  
=A\cap \overline{B\cup C}  
=A\setminus(B\cup C).  
]

Равенство доказано.

Для диаграммы Эйлера-Венна надо нарисовать три круга (A,B,C).  
На обеих диаграммах заштриховывается одна и та же область: **та часть круга (A), которая не принадлежит ни (B), ни (C)**.

## Задание 11

Построить релейно-контактную схему с функцией проводимости:

[  
(x\to y)\to x'(y\vee z).  
]

Преобразуем функцию.

[  
x\to y=x'\vee y.  
]

Тогда

[  
F=(x'\vee y)\to x'(y\vee z).  
]

[  
F=(x'\vee y)'\vee x'(y\vee z).  
]

По закону де Моргана:

[  
(x'\vee y)'=xy'.  
]

Следовательно,

[  
F=xy'\vee x'(y\vee z).  
]

Раскроем скобки:

[  
F=xy'\vee x'y\vee x'z.  
]

**Схема:** параллельно соединяются две ветви:

1. первая ветвь: контакты (x) и (y') последовательно;
    
2. вторая ветвь: контакт (x') последовательно с параллельным соединением (y) и (z).
    

Можно записать так:

[  
F=xy'\vee x'(y\vee z).  
]
